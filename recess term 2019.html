Loops

It is rarely useful that a computer performs each operation only once; we usually need it to repeat what it does. To do so, we need loops.

for-loops

The most common type is a for-loop, which is usually used to execute some part of the code a predetermined number of times.



example 1

fruits = ["apple", "banana", "cherry"]

for x in fruits:

  print(x)



Example 2

for x in "banana":

  print(x)



while-loops

The most common type is a for-loop, which is usually used to execute some part of the code a predetermined number of times.



example 1

n = 4

i = 0

while i < n:

  print(i)

  i += 1



functions

def max1(x, y):

  #Takes comparable values x and y as arguments.

  #Returns the larger of the two, implemented with if-else.

  if x > y:

     return x

  else:

     return y



def max2(x, y):

  #Takes comparable values x and y as arguments.

  #Returns the larger of the two, implemented with if (no else).

  if x > y:

     return x

  return y



a = 6

b = 2

print("Max1:", max1(a, b))

print("Max2:", max2(a, b))
